Utiliser Tor

Vidalia facilite le contrôle de votre instance Tor en permettant aussi bien son démarrage et son arrêt que la surveillance de son état et l'analyse des causes d'un éventuel arrêt impromptu.

Démarrer et arrêter Tor

Pour lancer Tor, sélectionnez Démarrer dans le menu de Vidalia situé dans la zone de notification ou appuyez sur Ctrl+S. L'icône de Vidalia de la zone de notification change lorsque Tor démarre : l'oignon barré par un X rouge est remplacée par un oignon vert.

Si Vidalia ne parvient pas à démarrer Tor, il affiche un message d'erreur pour indiquer qu'un problème est survenu. Vous pouvez également consulter le Journal des messages pour voir si Tor n'a pas donné d'autres informations sur le problème.

Pour arrêter Tor, sélectionnez Arrêter dans le menu de Vidalia situé dans la zone de notification ou appuyez sur Ctrl+T. L'icône de Vidalia de la zone de notification change lorsque Tor s'arrête : l'oignon vert est remplacé par un oignon barré par un X rouge.

Si Vidalia ne parvient pas à arrêter Tor, il affiche un message d'erreur pour indiquer qu'un problème est survenu. Vous pouvez également consulter le Journal des messages pour voir si Tor n'a pas donné d'autres informations sur le problème.

Surveiller l'état de Tor

Vidalia indique l'état de Tor par le biais d'une icône située dans la zone de notification ou dans la barre des tâches. Le tableau suivant précise la correspondance entre cette icône et l'état de Tor.

Tor est arrêté. Sélectionnez Démarrer dans le menu de Vidalia pour lancer Tor.
Tor est en cours d'initialisation. Vous pouvez consulter le Journal des messages pour connaître les informations sur l'état de Tor lorsqu'il démarre.
Tor est en cours de fonctionnement. Si vous voulez savoir ce qu'il fait, vous pouvez consultez le Journal des messages que Tor envoie à Vidalia. Si vous voulez arrêter Tor, sélectionnez Arrêter dans le menu de Vidalia.
Tor est en cours d'arrêt.

Si Tor s'arrête de manière inattendue, Vidalia affiche l'icône de l'oignon noir barré d'un X rouge et montre un message d'erreur pour vous signaler le problème. Vous pouvez également consulter le Journal des messages pour avoir davantage de details sur les problèmes rencontrés par Tor avant de se terminer.
